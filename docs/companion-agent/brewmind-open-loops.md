## How to use this file

- **Add an item** when a BrewMind thread opens and has no resolved governed artifact.
- **Annotate evidence tier** ([A], [B], [C]) on any claim about budget, partners, or commitments.
- **Note tensions** inline using the pattern: `Tension: [source-1] says X; [source-2] says Y. Unresolved as of YYYY-MM-DD.`
- **Remove an item** when it is resolved and either (a) closed with no action needed, or (b) promoted to a proposal and approved into governed state.
- **Do not** copy Supabase captures verbatim as if they are canonical facts — they are Tier C until verified.
