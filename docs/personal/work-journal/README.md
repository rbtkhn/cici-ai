# Work Journal

Daily learning log for Xavier's AI skill-building practice.

## How to use

1. Run `/daily-task` at the start of each session to get today's task card and create a journal stub
2. Complete the task (15–30 min)
3. Fill in the journal entry: what you did, what you learned, your reflection
4. Commit the entry when done

## File naming

`YYYY-MM-DD.md` — one file per day.

Build the skills to teach Cici to manage the BrewMind business and website using Claude skills, governed state, agents, and automation.
