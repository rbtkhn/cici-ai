# Prepared Context — Archived

This directory holds **expired or superseded prepared-context artifacts**.

## Purpose

When a synthesis, session artifact, or pending-review item is no longer active, move it here instead of deleting it. This preserves the paper trail.

## What ends up here

- Synthesis documents that were superseded by newer versions
- Session artifacts that contained something worth keeping (but the session is over)
- Pending-review items that the owner reviewed and decided not to promote
- Any prepared-context artifact that is no longer actionable but might be historically useful

## Naming

Retain original filenames. Optionally prefix with `archived-`:

```
archived-20260411-brewmind-capture-themes.md
```

## Retention

Indefinite. Low review priority. Clean up annually if the directory becomes unwieldy.

See `docs/prepared-context-doctrine.md` for the full doctrine.
